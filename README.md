##正文
-*这道题是有关摩擦力的,众所周知,在摩擦力的作用下加速度会和速度成正比,导致速度逐渐降低至一个极限值；*
-*用公式表达就是 $dv/dt=a-bv  ,当速度降低至a/b时加速度为0,此后速度恒定,可见,速度曲线应该是一个逐渐下降逼近一个临界值.*
-*用欧拉法解决该题就是将微分方程化为差分方程,即
 $$ v(t+detat)=v(t)+(a-b*v(t))*detat*
-*选择参数a=10,b=1,和初值为v(t=0)=100*
